# open-depth


## Prerequisites

    CUDA 8.0
    python 3.6.x
    pytorch 0.3.0

## Installation

    git clone https://github.com/yokattame/open-depth.git 
    cd open-depth/depth/networks/correlation_package
    ./make.sh
    cd ../../..
    python setup.py clean
    python setup.py install

## Examples

    ./scripts/train_baseline.py
